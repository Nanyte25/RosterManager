# Installation

### Install meteor

```
curl https://install.meteor.com/ | sh
```

### Clone this repo

```
git clone https://github.com/Nanyte25/RosterManager.git
```

### Install dependencies

```
meteor npm install 
```

### Run application

```
meteor run
```
